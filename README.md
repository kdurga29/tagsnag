Track Myntra prices. Get alerts when they drop.

TagSnag is a full-stack web application that allows users to track Myntra product prices in real time.  
Users can paste a Myntra product link, and TagSnag automatically scrapes the product price, stores price history, and sends an email alert when the price drops.

##Features
- User authentication (Signup / Login / Logout)
- Track Myntra product prices using product links
- Automatic background price monitoring
- Email alerts when price drops
- Price history stored in MongoDB
- Stop tracking by deleting products
- Light / Dark mode UI
- Fully responsive modern UI

##Tech Stack
Frontend
- Next.js (App Router)
- React
- Tailwind CSS
- Lucide Icons
  
Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- Puppeteer (for scraping)
- Nodemailer (email alerts)
- JWT Authentication
